# Бот учёта расходов

## Категории:
- 🛒 Продукты
- ☕ Кафе
- 💊 Здоровье
- 📢 Реклама
- 🏛 Налоги
- 🎁 Донат
- 📦 Другие траты

## Как запустить:

### 1. Создай бота в Telegram:
- Напиши @BotFather в Telegram
- Команда /newbot
- Придумай имя и username для бота
- Скопируй токен

### 2. Установи зависимости:
```
pip install -r requirements.txt
```

### 3. Вставь токен:
Открой bot.py и замени YOUR_BOT_TOKEN_HERE на свой токен.
Или задай переменную окружения: BOT_TOKEN=your_token

### 4. Запусти:
```
python bot.py
```

## Функции:
- Добавить расход по категории
- Добавить заметку к расходу
- Статистика за всё время
- Статистика за текущий месяц
- Сброс всех данных
